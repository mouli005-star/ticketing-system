# ticketing-system
It is to create an ticketing system to re-solve the student issuess in NIT calicut works as below mentioned :
1. Frontend : for users to Intereact (like riase an issuess and check the status of them)
2. Backend : Contains the Logic to perform UI activities and sends the data to DB
3. Data Base : Stores the User Issue details and Login Credentials
4. Agent : will send the Issue Information to the respective People like electrical problem to electrician, Network problem to CNC etc.
